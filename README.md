# c-cli-gcc-ninja-multi-file-hello-world

## Description
This is a POC project to demonstrate 
ninja as a build tool.

## Tech stack
- c
- gcc
- ninja

## Docker stack
- alpine:edge

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
